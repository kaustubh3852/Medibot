# 🏥 MediBot - AI Healthcare Chatbot

A conversational AI healthcare assistant built with Flask and Google Gemini API that provides general health information and wellness tips.

## 🚀 Features
- AI-powered health query responses
- Real-time chat interface
- Built with Google Gemini 2.0

## 🛠️ Tech Stack
- **Backend:** Python, Flask, REST API
- **AI:** Google Gemini API
- **Frontend:** HTML, CSS, Bootstrap, JavaScript

## ⚙️ How to Run

1. Clone the repository
   git clone https://github.com/kaustubh3852/Medibot.git

2. Install dependencies
   pip install -r requirements.txt

3. Add your Gemini API key in .env file
   GEMINI_API_KEY=your_key_here

4. Run the app
   python app.py

5. Open browser
   http://127.0.0.1:5000
